Introdução a CSS

Aula 4 sobre CSS

Existem 3 formas de utilizar o CSS:

Inline

A 1ª é a Inline, que é quando vou direto na tag do arquivo HTML e insiro ali um atributo de estilo (style) e dai consigo passar o que quero, exemplo: estilização desse paragrafo.

Esta não é uma boa prática

Interna

A 2ª forma é a Interna , onde vai ficar dentro do arquivo HTML, mais especificamente dentro da tag head, onde será criada uma tag chamada style e dentro dela passo outras tags, mencionada-as e adicionando o estilo que quero criar, exemplo: estilização da cor de fundo e da cor do Titulo "Introdução a CSS"

Externa

A 3ª e ultima forma é externa onde será criado um outro arquivo, fora do HTML e dai vou chamar esse arquivo dentro do HTML, isso se chama separação de responsabilidades
Ai, no meu arquivo HTML dentro da minha tag head vou chamar o arquivo do CSS atraves do style, e dentro dele vou usar uma outra tag chamada link vou colocar um atributo chamado rel, colocando entre as aspas um atributo que indica que estou usando um link de estilo: link rel="stylesheet", em seguida vou incluir mais um atributo, que vai passar o caminho do meu arquivo CSS: href="style.css" ficando ao final: link rel="stylesheet" href="style.css"

Feito isso, terei dois modos de passar a estilização, atraves da classe onde irei nas tags que quero estilizar, vou por dentro delas o atributo class="nomeDoParametro", depois vou no arquivo CSS e vou estilizar do seguinte modo: vou chama-lo com .nomeDoParametro{} e dentro das chaves vou passando os atributos que quero, como: color: ; e posso passar essa classe em outras tags que eu quiser.

O outro modo de passar a estilização é atraves do id, porem esse é indicado parametriza-lo com a finalidade de usar uma única vez, para usar vou na tag do HTML que quero, e vou por: id="nomeQueQuiser" e no arquivo CSS vou chama-lo: #nomeQueQuiser{ color:"cor x"}
